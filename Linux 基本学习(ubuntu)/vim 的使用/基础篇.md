# 一、安装
1. 输入 vim
2. 按照提示做就行了

# 二、使用
1. 编写一个文件 "vim 文件名称"
2. 写文件 "i" 或者 "s"
# 三、退出
1. 保存退出：<br>
      先按 "ESC" 然后输入 ":wq"
2. 不保存退出:<br>
      先按 "ESC" 然后输入 ":q!"<br>
# 三、配置 vim 
1. 在终端输入 “vim~/.vimrc”
2. 设置:<br>
(1). 改变代码的颜色 "syntax on"<br>
(2). 设置行号 "set number"<br>
(3). 支持鼠标 "set mouse=a"<br>
(4). 按下 Tab 键时，Vim 显示的空格数 "set tabstop=2"<br>
(5). 按下回车键后，下一行的缩进会自动跟上一行的缩进保持一致 "set autoindent"<br>
(6). 高亮显示当前行 "set cursorline"<br>
(7). vim命令自动补全 "set wildmenu"<br>
(7). 显示光标位置 "set ruler"
# 四、移动光标
1. 移动光标 "行号gg"
2. 删除一行 "cc"
3. 删除两行 "c2c"
4. 恢复 "u"
5. 复制 "p"
6. 一次向右跳一个单词的距离 "w"
7. 一次向左跳一个单词的距离 "B"
