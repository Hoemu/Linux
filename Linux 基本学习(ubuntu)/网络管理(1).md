## Linux 配置IP地址的方法
1. ifconfig命令临时配置
    
    一般我们使用**ifconfig**这个命令用来查看IP地址，但是也可以用于**临时修改** **IP**和**子网掩码**，命令如下：
    ```
    ifconfig eth0 192.168.254.200 netmarsk 255.255.255.0
    ```
    在查看，会出现的
    ```
        inet addr 是IP地址
        Bcast 广播地址
        Mask 子网掩码
    ```

2. setup工具永久配置IP地址

    setup是一个工具，如果没有这个工具的话，那需要下载这个工具，具体具体操作和图形界面操作差不多

3. 修改网络配置文件
    
    在ubuntu中，网络配置文件通常在 **/etc/network/interfaces** 这个文件中，如果要永久修改的化，可以直接修改这个文件
    
4. 图形界面配置IP地址

    使用有图形界面的Linux可直接在图形界面上查看和修改，一般在设置里面可以找到

## 关闭和启动网卡

* ifdown 网卡设备名

    &禁用该网卡设备

* ifup 网卡设备名

    &启动该网卡设备

#### 有些Linux中可能不能使用这连个命令，那么就需要下载工具了

## 查询网络状态

* netstat 选项
    
    选项：
    
        -t:列出TCP协议端口
        -u:列出UDP协议端口
        -n:不适用域名或服务名，而使用IP地址和端口号
        -l:仅列出在监听状态网络服务
        -a:列出所有的网络连接
    
    当然我们可一起输入查看所有情况
    ```
    netstat -tuln
    ```

    查看有多少个人连接到我们的端口
    
    ```
    netstat -an | grep ESTABLISHED

    netstat -an | grep ESTABLISHED | wc -l
    ```
* 查看网关
    ```
    netstat -rn

    nroute -n

    结果会出现gateway

    geteway为网关

    ```
    临时修改网关
    ```
    route add default gw 192.168.1.1
    (要在同一个网段中)

    ```
* 查看nslookup

    一般可用来查看DNS
    ```
    nslookup www.baidu.com
    (Server为DNS)

    quit退回
    exit退出
    ```

* telnet命令

    telnet命令为远程管理于端口探测命令

    使用方法
    ```
    telnet [域名或IP] [端口]
    
    如下：

    telnet 192.168.0.252 80
    ```

* tranceroute
    
    tanceroute是路由跟踪命令

    tranceroute [选项] IP或域名

    选项:
    ```
        -n 使用IP，不适用域名，速度更快
    ```
    
* wget命令

    这是一个下载命令

    wget [地址]

    连接服务器，然后进行下载
